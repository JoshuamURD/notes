# Course details  
This is a Udemy course by Colt Steele who is also a Youtuber. The course covers several topics in web development, security and thinking. The course assumes zero web development knowledge:  
- HTML   
- CSS  
	- Bootstrap  
- Javascript  
	- Express.js  
	- React  
- MongoDB
